
This project contains materials for a hands-on tutorial on
Microscopy Image Analysis.

The material is initially intended for the MICCAI 2010 Tutorial:

   Microscopic Image Analysis and Visualization
   Beijing, Sept 24, 2010
   http://penglab.janelia.org/proj/miccai2010/About.html

but we anticipate that the material can be reused and be posted
online to provide a wider access.
